# Overview for Mint 2050

Move tiles to add and create the 2048 tile.

## Get powers from 
- Creating 2048 tiles
- Making high scoring moves
- Doubling highest tile

## Mint (powers mode)
	Multiply
		turn a 2 into a 4
	Split
		halves a tiles number
	New Tile
		create new tile
	Freeze
		Prevent a tile from merging

## Cash Mode
	Currency is the player's score
